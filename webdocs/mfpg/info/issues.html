Known Issues

Noteflight musicXML Export

Noteflight exports musicXML files containing tuplets in an order not supported by the mx library. Currently it cannot successfully parse scores directly exported to musicXML by Noteflight.

This can be solved by exporting the score as a MIDI file (or some other accepted export format by Noteflight), and then using a converter to convert it to musicXML. This issue should be fixed in future updates where a converter is automatically used in the program when the file format is not musicXML.
