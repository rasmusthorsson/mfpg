Available Guides

GUI Guide

The GUI Guide gives a brief overview of the different GUI components and how to use them.

CLI Guide

The CLI Guide gives a brief overview of how to use the commandline interface and the various available options.

DSL Guide

The DSL Guide describes how to define MFPG DSL files. A DSL file is an important component to customize fingering position generation.

Notemap Builder Guide

The Notemap Builder Guide shows a few small examples of how to define your attribute value calculations. These attribute values are required for any attribute used in the DSL file.

Simple Example

The Simple Example Guide is a simple tutorial on how to use the program.
